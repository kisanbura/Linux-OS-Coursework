 Overview
This technical journal documents the configuration, hardening, monitoring, and performance evaluation of a **headless Linux server** administered remotely via SSH from a dedicated workstation.  

The project emphasises:
- Command-line proficiency  
- Secure remote administration  
- Performance evaluation under varied workloads  
- Reflective learning through weekly updates
